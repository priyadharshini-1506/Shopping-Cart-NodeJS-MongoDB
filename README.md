# 🛒 Shopping Cart Website

A full-stack e-commerce application developed using **Node.js** and **MongoDB** that allows users to browse products, create accounts, manage shopping carts, and place orders through a seamless checkout experience.

🚀 Features

* User Registration and Login
* Product Listing and Browsing
* Add to Cart and Remove from Cart
* Secure Checkout Workflow
* Payment and Order Confirmation
* Customer Information Management
* Order Tracking and Management

🗄️ Database Integration

MongoDB was integrated as the backend database to store and manage:

* User registration details
* Authentication data
* Customer information
* Product information
* Shopping cart data
* Order details and purchase history

🛠️ Technologies Used

* **Backend:** Node.js
* **Database:** MongoDB
* **Programming Language:** JavaScript
* **Frontend:** HTML, CSS
* **Package Management:** npm
